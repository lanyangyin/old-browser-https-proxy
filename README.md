- 由于psp3000的浏览器无法进行ssl握手导致无法进入https网页而产生的项目

- 本项目由[DeepSeek](https://chat.deepseek.com/share/5wosfylaqxsi8tbamq)完成

- 项目参考了[snacsnoc/windows98-ie-proxy](https://github.com/snacsnoc/windows98-ie-proxy)
- 因此使用[GPL v3](LICENSE)协议

## 安装
```bash
git clone https://github.com/lanyangyin/old-browser-https-proxy.git
cd old-browser-https-proxy
pip install -r requirements.txt
```

## 用法
```bash
python universal_proxy_with_favorites.py
```
